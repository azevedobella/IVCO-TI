# IVCO-TI

Repositório com os shapefiles que sustentam o Índice de Vulnerabilidade diante do Crime Organizado: Territórios Indígenas.

## Objetivo
Organizar código, documentação e infraestrutura relacionados ao índice.

## Estrutura inicial proposta
- README.md — apresentação do projeto
- .gitignore — padrões gerais

## Como iniciar localmente
1. Clonar o repositório:
   git clone git@github.com:azevedobella/IVCO-TI.git
2. Entrar na pasta:
   cd IVCO-TI
3. Criar os arquivos (se ainda não estiverem no remoto) e fazer o commit inicial:
   git add .
   git commit -m "chore: inicializar repositório"
4. Enviar para o remoto (cria o branch main se for o primeiro push):
   git push -u origin main
